## Deutsch

### Was ist diese App?

Eine kostenlose, mehrsprachige Lern-App für Kinder, die das Lesen und Stellen einer analogen Uhr sowie das Schreiben von Zahlen üben möchten. Die App wurde für den privaten Gebrauch entwickelt und läuft direkt im Browser — ohne Installation, ohne Registrierung, ohne Werbung.

### Übungsmodi

- **Uhr lesen** — die angezeigte Zeit aus vier Antworten auswählen
- **Zeiger stellen** — Zeiger per Drag & Drop oder Schieberegler auf die gewünschte Zeit stellen
- **Text → Uhr** — eine ausgeschriebene Zeit auf der Uhr einstellen
- **Uhr → Satz** — Wörter in die richtige Reihenfolge bringen um die gezeigte Zeit zu beschreiben
- **🔢 Zahlen schreiben** — eine angezeigte Zahl (0–24) als Wort eintippen, mit bis zu 3 Versuchen

### Funktionen

- **4 Sprachen** — Deutsch, Italienisch, Englisch, Japanisch
- **3 Schwierigkeitsstufen** — Einfach (volle / halbe Stunden), Mittel (Viertelstunden), Schwer (5-Minuten-Schritte)
- **24-Stunden-Format** mit ausgeschriebenen Zahlen
- **Automatische Zahlen-Übung** — nach jeder 5. Uhr-Aufgabe erscheint automatisch eine Zahlen-Übung
- **Mehrere Profile** — jedes Kind hat sein eigenes Profil mit separaten Statistiken
- **Detaillierte Statistiken** — pro Sprache, pro Modus und pro Zahl (0–24)
- **Adaptives Lernen** — schwierige Zeiten werden häufiger wiederholt
- **Lernpfad** mit 9 Stufen und 11 Abzeichen
- **Tagesaufgabe** — 5 Aufgaben pro Tag
- **Ton-Feedback** und **Sprachausgabe** (bei falscher Antwort wird die richtige Lösung vorgelesen)
- **Dunkelmodus**
- **Offline-fähig** (PWA — installierbar auf Android und iOS)

### Technologie

Reines HTML / CSS / JavaScript — keine externe Bibliothek, kein Framework. Läuft auf jedem Gerät mit einem modernen Browser. Fortschritt wird lokal im Browser gespeichert (localStorage).

### Nutzung

Die App ist unter https://dimitrisdisandro.github.io/uhr-app erreichbar. Auf Android/iOS kann sie über «Zum Startbildschirm hinzufügen» wie eine native App installiert werden.

### Hinweis

Diese App wurde für den privaten Gebrauch entwickelt und wird ohne Gewähr bereitgestellt. Sie dient ausschliesslich zu Lernzwecken. Für allfällige Fehler oder Ungenauigkeiten wird keine Haftung übernommen.

## English

### What is this app?

A free, multilingual learning app for children who want to practise reading and setting an analogue clock and writing numbers. The app was developed for private use and runs directly in the browser — no installation, no registration, no advertising.

### Exercise modes

- **Read the clock** — choose the correct time from four answers
- **Set the clock** — set the hands using drag & drop or sliders
- **Text → Clock** — set the clock to a written-out time
- **Clock → Sentence** — arrange words in the correct order to describe the shown time
- **🔢 Write numbers** — type a shown number (0–24) as a word, with up to 3 attempts

### Features

- **4 languages** — German, Italian, English, Japanese
- **3 difficulty levels** — Easy (full / half hours), Medium (quarter hours), Hard (5-minute intervals)
- **24-hour format** with numbers written out in full
- **Automatic number exercise** — after every 5 clock tasks a number exercise appears automatically
- **Multiple profiles** — each child has their own profile with separate statistics
- **Detailed statistics** — per language, per mode and per number (0–24)
- **Adaptive learning** — difficult times are repeated more often
- **Learning path** with 9 levels and 11 badges
- **Daily task** — 5 exercises per day
- **Sound feedback** and **text-to-speech** (on wrong answer the correct solution is read aloud)
- **Dark mode**
- **Offline capable** (PWA — installable on Android and iOS)

### Technology

Pure HTML / CSS / JavaScript — no external library, no framework. Runs on any device with a modern browser. Progress is saved locally in the browser (localStorage).

### Usage

The app is available at https://dimitrisdisandro.github.io/uhr-app. On Android/iOS it can be installed as a native app via «Add to Home Screen».

### Disclaimer

This app was developed for private use and is provided without warranty. It is intended for educational purposes only. No liability is accepted for any errors or inaccuracies.

## 日本語

### このアプリについて

アナログ時計の読み方と合わせ方、そして数字の書き方を練習したい子どもたちのための、無料の多言語学習アプリです。個人使用のために開発されており、インストール・登録・広告なしでブラウザ上で直接動作します。

### 練習モード

- **時計を読む** — 4つの選択肢から正しい時刻を選ぶ
- **針を合わせる** — ドラッグ＆ドロップまたはスライダーで針を設定する
- **テキスト→時計** — 文字で書かれた時刻を時計に合わせる
- **時計→文章** — 表示された時刻を説明する言葉を正しい順番に並べる
- **🔢 数字を書く** — 表示された数字（0〜24）を言葉で入力する（最大3回挑戦）

### 機能

- **4言語** — ドイツ語、イタリア語、英語、日本語
- **3段階の難易度** — かんたん（整数・30分）、ふつう（15分単位）、むずかしい（5分単位）
- **24時間形式**（数字をすべて言葉で表記）
- **自動数字練習** — 5問の時計問題ごとに自動的に数字練習が表示される
- **複数プロフィール** — 子どもごとに個別のプロフィールと統計
- **詳細な統計** — 言語別・モード別・数字別（0〜24）
- **適応学習** — 苦手な時刻が優先的に出題される
- **学習パス** — 9段階・11種類のバッジ
- **今日の課題** — 1日5問
- **音のフィードバック**と**読み上げ機能**（不正解時に正解が読み上げられる）
- **ダークモード**
- **オフライン対応**（PWA — AndroidとiOSにインストール可能）

### 技術

純粋なHTML / CSS / JavaScript — 外部ライブラリやフレームワークなし。モダンブラウザを搭載したあらゆるデバイスで動作します。進捗はブラウザのlocalStorageにローカル保存されます。

### 使い方

アプリは https://dimitrisdisandro.github.io/uhr-app でご利用いただけます。Android/iOSでは「ホーム画面に追加」からネイティブアプリとしてインストールできます。
